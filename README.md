# 🌌 Overview

**OfferPilot** is a modern AI-driven career platform designed for students, developers, and professionals preparing for top tech companies.

It combines:

- 🎯 ATS Resume Optimization
- 🤖 AI Mock Interviews
- 💻 Coding Practice
- 📈 Interview Analytics
- 🧠 Skill Tracking
- 📄 Resume Builder
- 🔥 Premium Landing Experience
- 🌙 Fully Animated Dark/Light UI

into one beautiful production-grade application.
